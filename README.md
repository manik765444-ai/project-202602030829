# Memory Master
================

## Description
A simple yet addictive HTML memory card game to test your brain cells. Flip, match, and repeat - can you master the Memory Master?

## Features
* **Easy to play**: Intuitive interface and simple gameplay make it accessible to players of all ages.
* **Randomized cards**: Cards are shuffled randomly on each new game, ensuring a unique experience every time.
* **Quick reflexes**: Timed gameplay adds an extra layer of challenge - can you make all the matches before the clock runs out?
* **No distractions**: Clean, minimal design keeps the focus on the game.

## Installation and Usage
No installation needed! Simply clone the repository or download the zip file, then open `index.html` in your browser to start playing.
```bash
git clone https://github.com/manik765444-ai/project-202602030829.git
cd project-202602030829
open index.html
```
Alternatively, you can play the game directly from [GitHub Pages](https://manik765444-ai.github.io/project-202602030829/).

## Technologies
* **HTML**: The game is built using plain old HTML, making it lightweight and easy to maintain.
